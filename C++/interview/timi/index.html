    腾讯天美游戏客户端一面

    腾讯天美游戏客户端开发， 电话面(30min)

    1.C++基础内置类型大小 int 4; long 4或8 ;long long 8 ;float 4 ; double 8; 2.值传参和引用传参的区别

    3.值传参和引用传参传递效率的区别, 对于内置类型也是引用传参的效率高吗

    4.int最小值, 最大值的二进制表示

    5.有符号整数和无符号整数的右移操作

    6.无符号整数最大值, 右移一位最高位是1还是0

    7.知道上面的移位概念吗

    8.C语言 位段/位域

    9.const int 和 int const const int 指针指向的是常量，int const 指针是常量 10.const修饰函数体

    11.const是否可以修饰全局函数或静态成员函数

    12.void*的用法

    13.void* 类型是否能做算术运算

    14.普通函数指针和成员函数指针的区别

    15.什么是字节序, 字节序的定义, 什么情况下需要注意字节序, 如何判断本机的字节序

    16.new 和 malloc 的区别 new 在分配内存时会调用构造函数(如果是对象),它返回的是指定类型的指针，无需显示转换。 malloc仅分配内存，不调用构造函数，返回void* 17.知不知道什么是placement new

    18.什么是内存对齐, 为什么要进行内存对齐 为提高内存访问效率，数据是某个对齐单位的整数倍，比如int(4 byte)它的起始地址就是4的倍数; 但是对齐可能会浪费内存空间，比如一个结构体中有个char，还有个int，char后面可能会填充三个字节的空隙。

    19.C++中一个空的class大小 通常是一个字节，即使class是空的在创建对象时每个对象也需要分配一个地址。

    20.含有一个虚函数成员的class大小 含有虚函数的class通常比空的class要大，里面有个虚函数表存储虚函数地址，每个对象会有一个指针指向虚函数表。64位系统中vptr通常为8字节，64/8bit=8字节

    21.讲讲智能指针的作用 自动管理内存，防止内存泄漏，无需手动delete 管理资源，有shared_ptr ,unique_ptr

    22.左值和右值的定义, 左值引用可以指向右值吗

    左值在程序运行期间有存储地址的表达式，他可以出现在赋值运算符左侧，也可以放在右侧，右值通常是临时值，字面量，表达式的结果，程序运行期间没有明确存储地址 普通左值引用无法绑定右值，但是const左值引用可以绑定，右值的生命周期也会延长到引用的生命周期结束 

    int a = 10;
int& ref = 10; // 错误：普通左值引用不能绑定右值

const int& ref = 10; // 合法：const 左值引用可以绑定右值
std::cout << ref << std::endl; // 输出 10
    23.移动语义的目的 将资源从一个对象转移到另一个对象，避免直接赋值的深拷贝过程导致性能下降， 
    vector<int> v2 = move(v1); // 移动语义，转移资源
    移动后原来对象的指针会指向nullptr 移动后的原对象仍然是有效的，但它的资源已经被转移，通常不再持有任何有意义的数据。原对象可以被安全销毁或重新赋值。

    24.C++的class和struct有什么区别

    25.讲一下数组和链表的区别 数组连续存储，支持随机访问，链表在内存中非连续存储，每个节点通过指针指向下一个节点，数组在访问时方便，但插入删除会移动大量数据，链表在插入删除很友好 数组在创建时需要分配好固定的内存，链表内存可以是动态分配。

    26.如何找到一个链表的中间节点

    要是链表是双向链表可以 用两个指针，一个指向头节点，一个指向尾节点，两个按照同样的速度向中间移动

    单向链表也可以快慢指针，开始一个fast，一个slow指针指向头节点，快指针每次移动两步，慢指针移动一步，快指针到末尾时，慢指针就走到中间。

    27.STL的sort函数由什么组成 它是个混合排序算法，有快排(O(nlogn)),数据量较小时就用插入排序(O(n^2)) 某些情况使递归深度太大，避免陷入快速排序的最坏情况会切换到堆排序。 sort()是不稳定排序，stable_sort()是稳定的

    28.快排的基本思路, 什么情况出现最坏情况 选一个基准(比如要排序段的第一个值为索引)，将大于基准的值放到基准后面，小于基准的值放到基准前面，然后递归调用这个函数对前半段和后半段快排。

    29.进程和线程的区别, 进程有哪些资源, 线程有哪些私有资源 进程是操作系统分配资源的基本单位，每个进程有独立的内存空间和系统资源 线程是进程中的一个执行单元，是CPU调度的基本单位
